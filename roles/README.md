# Public Roles

This directory contains generic role overlays that extend the public core `AGENTS.md` contract.

## Roles

- `architect`: system and implementation planning.
- `coordinator`: sequencing, delegation, and handoff management.
- `developer`: scoped implementation work.
- `documenter`: durable documentation and runbooks.
- `operator`: approved operational commands and maintenance.
- `reviewer`: findings-first review of code, plans, and artifacts.

## Boundaries

- Keep examples synthetic and reusable.
- Do not include private identities, hostnames, workspace paths, credentials, transcripts, or customer details.
- Runtime binding and deployment-specific behavior belong outside this public role layer.
